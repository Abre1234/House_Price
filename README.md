# 🏠 House Price Data Analysis Project

## 📌 Overview
This project focuses on analyzing a structured housing dataset to understand key factors influencing house prices. The workflow includes data cleaning, preprocessing, exploratory data analysis (EDA), and visualization to extract meaningful insights from the data.

The goal is to practice real-world data analysis skills such as data cleaning, handling missing values, detecting patterns, and communicating insights through visualizations.

---

## 🎯 Objectives
- Clean and preprocess raw housing data
- Handle missing values, duplicates, and inconsistencies
- Perform exploratory data analysis (EDA)
- Identify patterns and relationships between variables
- Visualize key insights for better understanding

---

## 🛠️ Tools & Technologies
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Project Workflow

1. **Data Loading**
   - Import dataset and inspect structure

2. **Data Cleaning**
   - Handle missing values
   - Remove duplicates
   - Fix inconsistent data types

3. **Exploratory Data Analysis (EDA)**
   - Summary statistics
   - Correlation analysis
   - Feature distribution analysis

4. **Data Visualization**
   - Price distribution plots
   - Relationship between variables
   - Heatmaps and scatter plots

5. **Insights Extraction**
   - Key factors affecting house prices
   - Patterns and trends in the dataset

---

## 📈 Key Insights
- House prices vary significantly based on structural and location-related features
- Certain variables show strong correlation with price
- Data quality plays a major role in reliable analysis results
